# AGENTS.md

## Overview
Landscape It is an AI-guided landscaping app for everyday homeowners.
Build consumer-facing product experiences only; do not expose internal tech, scaffolding, or implementation details in the UI.
Use `docs/prds` for product requirements and `.agents/` for agent references and repo workflow notes.

## Stack
- Bun workspaces
- SvelteKit 2 + Svelte 5
- Tailwind CSS v4
- Better Auth for authentication
- Drizzle ORM + PostgreSQL
- Polar for credit-pack billing
- S3-compatible object storage for uploads and generated images
- Docker Compose for local infrastructure
- mise for developer tasks
- Biome for formatting and linting
